Определение вертикального ритма

1. Размер шрифта и интерлиньяж по умолчанию. Задаются для body (html) в пикселях.
fz = 12px;
lh = 1.5 (То есть 12 * 1.5).
Затем остальным элементам задаем значения размера шрифта и высоты строки
в относительных единицах.
 p { fz = 1em;
     lh = 1.5em;
}

2. Определение отступов (вертикальный margin).
Большинство браузеров устанавливают верхний и нижний отступ в 1em
(то есть равные кеглю шрифта).Но чтобы поддерживать хороший ритм, отступы должны быть того же размера,
что и высота строки(интерлиньяж). Потому что расстояние между абзацами такое же, как и расстояние между
строками, что делает движение текста ровным и гармоничным.
p {
    font-size: 1em; /* эквивалентно 12px */
    line-height: 1.5em;
    margin: 1.5em 0;
}

3. Заголовки
h1 = 2em;
h1-lh = 1.5;

h2 = 1.5em;
h2-lh = 1.5;

h3 = 1.2em;
h1-lh = 1.5;
